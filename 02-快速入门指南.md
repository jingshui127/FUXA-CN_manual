# 🚀 FUXA 快速入门指南

---

## 📚 教程目标

本教程将演示 FUXA 最流行功能的基本用法。通过本教程，您将学会：

✅ 连接设备到 FUXA  
✅ 从设备推送数据到 FUXA  
✅ 构建实时 GUI SCADA/HMI/仪表板  
✅ 定义阈值并触发报警  

---

## 🔧 前置准备

在开始之前，您需要：

### 1️⃣ 安装 FUXA 服务器

请按照 [**安装和运行指南**](Installing-and-Running.md) 完成 FUXA 的安装。

### 2️⃣ 访问 FUXA 界面

安装完成后，FUXA UI 将通过以下 URL 访问：
```
http://localhost:1881
```

FUXA Web 服务器提供两个主要页面：

| 页面 | URL | 说明 |
|------|-----|------|
| **可视化界面** | http://localhost:1881/home | 终端用户查看界面 |
| **编辑器** | http://localhost:1881/editor | 项目设计和开发界面 |

## 步骤 1：连接设备并配置标签

### 🔌 支持的设备类型

FUXA 支持多种工业设备：

| 设备类型 | 协议 | 适用场景 |
|---------|------|---------|
| **Modbus** | RTU/TCP | 通用工业设备 |
| **Siemens S7** | S7 协议 | 西门子 PLC |
| **OPC UA** | OPC UA | 工业自动化标准 |
| **BACnet IP** | BACnet | 楼宇自动化 |
| **MQTT** | MQTT | 物联网设备 |
| **Ethernet/IP** | Allen Bradley | 罗克韦尔设备 |

### 📋 配置步骤

1. **获取设备凭证**
   - IP 地址
   - 端口号
   - 认证信息
   - 寄存器地址

2. **添加设备连接**
   - 进入编辑器的"连接"（Connections）页面
   - 添加设备
   - 配置通信参数

3. **配置标签**
   - 创建数据标签
   - 绑定到设备寄存器
   - 设置数据类型和单位

📖 **详细配置指南**：[设备和标签配置教程](HowTo-Devices-and-Tags.md)

## 步骤 4：配置报警系统 🚨

设置报警规则，当数据超出阈值时自动触发报警。

### 报警类型

| 报警级别 | 说明 | 应用场景 |
|---------|------|---------|
| **High High** | 高高报警 | 严重超标 |
| **High** | 高报警 | 超标预警 |
| **Low** | 低报警 | 低于下限 |
| **Message** | 消息报警 | 信息提示 |

### 报警功能

- ✅ 实时弹窗提示
- ✅ 声音报警
- ✅ 报警历史记录
- ✅ 短信/邮件通知

📖 **详细教程**：[如何设置报警](HowTo-setup-Alarms.md)

## 💡 实用技巧

### 🎯 最佳实践

1. **项目规划**
   - 先设计整体架构
   - 划分功能模块
   - 制定命名规范

2. **标签管理**
   - 使用清晰的标签名称
   - 添加描述和单位
   - 合理组织标签分组

3. **界面设计**
   - 保持界面简洁
   - 使用一致的配色
   - 考虑移动端适配

4. **性能优化**
   - 合理设置数据更新频率
   - 避免过多动画效果
   - 使用缓存减少请求

### ⚠️ 常见问题

| 问题 | 解决方案 |
|------|---------|
| 设备连接失败 | 检查 IP 地址、端口、网络连接 |
| 数据不更新 | 检查标签配置、设备通信状态 |
| 界面显示异常 | 清除浏览器缓存、检查浏览器兼容性 |
| 报警不触发 | 检查报警条件、阈值设置 |
