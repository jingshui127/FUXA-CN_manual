# ⚙️ FUXA 系统设置与使用技巧

---

## 📋 概述

本教程将介绍 FUXA 的系统设置配置和编辑器使用技巧，帮助您更高效地使用 FUXA。

---

## 🔧 系统设置

### 设置文件位置

服务器设置位于以下文件：

```
server\_appdata\settings.js
```

⚠️ **重要**：修改设置后必须重启服务器才能生效。

---

## 🔐 身份验证配置

### 启用身份验证

要启用和配置身份验证，请编辑 `settings.js` 文件中的以下参数：

```javascript
secureEnabled: true,            // 启用或禁用身份验证
secretCode: 'frangoteam751',    // 用于编码令牌的密钥
tokenExpiresIn: '1h'            // 令牌过期延迟 '1h'=1小时, 60=60秒, '1d'=1天
```

### 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| **secureEnabled** | Boolean | 启用或禁用身份验证 | `true` / `false` |
| **secretCode** | String | 用于编码令牌的密钥 | `'frangoteam751'` |
| **tokenExpiresIn** | String/Number | 令牌过期时间 | `'1h'` / `'1d'` / `60` |

### 默认用户

系统默认用户信息：

| 用户名 | 密码 | 权限 |
|--------|------|------|
| **admin** | **123456** | 管理员 |

⚠️ **安全提示**：首次使用后，请立即修改默认密码！

### 修改密码

1. 登录 FUXA 编辑器
2. 进入"设置"页面
3. 在用户管理中修改密码
4. 保存更改

## 💡 使用技巧

### 1️⃣ 快速复制组件

使用 **Ctrl + D** 快速复制选中的组件，比 Ctrl + C / Ctrl + V 更快捷。

**示例**：
1. 选中一个按钮组件
2. 按 Ctrl + D 复制
3. 重复按 Ctrl + D 创建多个副本

---

### 2️⃣ 精确对齐元素

使用 **Shift + 方向键** 精确移动元素，每次移动一个像素。

**示例**：
1. 选中需要对齐的元素
2. 使用 Shift + 方向键微调位置
3. 实现精确对齐

---

### 3️⃣ 快速旋转组件

使用 **Ctrl + 左/右方向键** 快速旋转组件，每次旋转 15°。

**示例**：
1. 选中需要旋转的组件
2. 按 Ctrl + Right 顺时针旋转
3. 按 Ctrl + Left 逆时针旋转

---

### 4️⃣ 批量操作

使用 **Ctrl + A** 全选所有元素，然后进行批量操作。

**示例**：
1. 按 Ctrl + A 全选所有元素
2. 按 Ctrl + G 分组
3. 移动整个组

---

### 5️⃣ 绘制标准线条

使用 **Shift + 绘制线条** 绘制水平、垂直或 45° 对角线。

**示例**：
1. 选择线条工具
2. 按住 Shift 键
3. 拖动鼠标绘制线条
4. 线条会自动约束到标准角度

---

### 6️⃣ 保持比例缩放

使用 **Shift + 鼠标调整大小** 保持元素宽高比例。

**示例**：
1. 选中需要调整大小的元素
2. 按住 Shift 键
3. 拖动调整大小
4. 宽高比例保持不变

---

### 7️⃣ 快速撤销和重做

使用 **Ctrl + Z** 撤销操作，**Ctrl + Y** 重做操作。

**示例**：
1. 执行一系列操作
2. 发现错误，按 Ctrl + Z 撤销
3. 如果撤销过多，按 Ctrl + Y 重做

---

### 8️⃣ 高效导航

使用 **Tab / Shift + Tab** 或 **Shift + O / P** 快速在元素之间切换。

**示例**：
1. 使用 Tab 选择下一个元素
2. 使用 Shift + Tab 选择上一个元素
3. 快速浏览和编辑元素

## 🔧 高级配置

### 自定义令牌过期时间

根据安全需求调整令牌过期时间：

```javascript
// 1 小时过期
tokenExpiresIn: '1h'

// 1 天过期
tokenExpiresIn: '1d'

// 60 秒过期
tokenExpiresIn: 60
```

### 修改密钥

定期更新密钥以提高安全性：

```javascript
// 原始密钥
secretCode: 'frangoteam751'

// 新密钥（建议使用强密码）
secretCode: 'YourStrongSecretKey123!@#'
```

⚠️ **注意**：修改密钥后，所有现有令牌将失效，用户需要重新登录。

## 🛡️ 安全最佳实践

### 1️⃣ 修改默认密码

首次部署后立即修改默认密码：

1. 登录系统
2. 进入用户管理
3. 修改 admin 密码
4. 保存更改

### 2️⃣ 使用强密钥

设置强密钥用于令牌编码：

- 使用至少 16 个字符
- 包含大小写字母、数字和特殊字符
- 定期更换密钥

### 3️⃣ 合理设置令牌过期时间

根据使用场景设置合理的令牌过期时间：

- 内网环境：可以设置较长时间（如 1 天）
- 公网环境：建议设置较短时间（如 1 小时）
- 高安全要求：设置更短时间（如 30 分钟）

### 4️⃣ 定期备份配置

定期备份 `settings.js` 文件：

```bash
# 备份配置文件
cp server/_appdata/settings.js server/_appdata/settings.js.backup
```

## 📊 性能优化

### 1️⃣ 合理使用分组

使用 **Ctrl + G** 将相关元素分组，提高编辑效率：

- 减少选择操作
- 便于批量移动
- 提高渲染性能

### 2️⃣ 避免过度使用动画

过多的动画会影响性能：

- 限制同时播放的动画数量
- 使用 CSS 动画代替 JavaScript 动画
- 在低端设备上禁用动画

### 3️⃣ 优化图表刷新频率

合理设置图表刷新频率：

- 实时数据：500-1000ms
- 趋势数据：2000-5000ms
- 历史数据：10000-60000ms

## ❓ 常见问题

### ❓ 修改设置后不生效

**可能原因**：
- 服务器未重启

**解决方案**：
```bash
# 重启 FUXA 服务器
npm restart
# 或
pm2 restart fuxa
```

### ❓ 无法登录

**可能原因**：
- 身份验证未启用
- 用户名或密码错误
- 令牌已过期

**解决方案**：
1. 检查 `secureEnabled` 设置
2. 确认用户名和密码正确
3. 重新登录获取新令牌

### ❓ 快捷键不工作

**可能原因**：
- 浏览器快捷键冲突
- 输入法激活
- 焦点不在编辑器

**解决方案**：
1. 关闭输入法
2. 点击编辑器区域获取焦点
3. 检查浏览器快捷键设置
