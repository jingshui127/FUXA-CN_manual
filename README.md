## 🛠️ 使用方法

### 查看文档

1. **直接打开** - 双击 `FUXA汉化文档_完整版.html` 即可在浏览器中查看
2. **搜索功能** - 使用顶部搜索框快速查找内容
3. **导航菜单** - 左侧导航栏可快速切换不同章节
4. **链接跳转** - 点击文档内的链接可跳转到相关章节

### 修改文档

如果需要修改文档内容：

1. 编辑对应的 `.md` 文件
2. 运行 `python 生成HTML.py` 重新生成HTML
3. 刷新浏览器查看更新

```bash
python 生成HTML.py
```
