# 📊 FUXA 图表控制教程

---

## 📋 概述

图表控制是 FUXA 中用于可视化历史数据和趋势的重要组件。通过图表，您可以直观地查看和分析设备数据的变化趋势。

---

## 🎯 添加图表控件

### 步骤 1：进入编辑器

打开 FUXA 编辑器。

### 步骤 2：选择视图

在编辑器中选择要添加图表的视图。

### 步骤 3：添加图表控件

从工具箱中拖拽 **Chart** 控件到视图上。

![添加图表控件](images/fuxa-chart.gif)

---

## 🔧 配置图表

### 配置折线图

在折线图中，您可以定义要附加到图表控件的图表。

![配置图表](images/setup-charts.png)

### 图表属性

| 属性 | 说明 |
|------|------|
| **图表名称** | 图表的唯一标识 |
| **标签** | 要显示的标签/变量 |
| **颜色** | 折线的颜色 |
| **线宽** | 折线的宽度 |
| **数据点** | 是否显示数据点 |
| **填充** | 是否填充折线下方区域 |

---

## 💡 使用示例

### 示例 1：温度趋势图

```javascript
// 配置温度标签
{
  tagId: 'temperature',
  label: '温度 (°C)',
  color: '#ff6b6b',
  lineWidth: 2,
  showPoints: true,
  fill: false
}
```

### 示例 2：多变量对比

```javascript
// 配置多个标签
[
  {
    tagId: 'temperature',
    label: '温度',
    color: '#ff6b6b'
  },
  {
    tagId: 'pressure',
    label: '压力',
    color: '#4ecdc4'
  },
  {
    tagId: 'flow',
    label: '流量',
    color: '#45b7d1'
  }
]
```

## 🎨 图表类型

### 折线图

- **用途**：显示数据随时间的变化趋势
- **特点**：适合连续数据
- **应用**：温度、压力、流量等

### 柱状图

- **用途**：比较不同类别的数据
- **特点**：适合离散数据
- **应用**：产量统计、能耗分析等

### 饼图

- **用途**：显示各部分占总体的比例
- **特点**：适合百分比数据
- **应用**：设备状态分布、能耗占比等

## 📚 最佳实践

### 1️⃣ 合理设置时间范围

- **实时监控**：设置较短的时间范围（如 5-10 分钟）
- **历史分析**：设置较长的时间范围（如 1 小时、1 天）
- **趋势预测**：使用足够的历史数据

### 2️⃣ 优化数据刷新率

- **高频数据**：使用较快的刷新率（如 1 秒）
- **低频数据**：使用较慢的刷新率（如 10 秒）
- **避免过度刷新**：防止系统负载过高

### 3️⃣ 使用合适的颜色

- **对比度**：确保颜色之间有足够的对比度
- **一致性**：相同类型的数据使用相同的颜色
- **可读性**：避免使用过于相似的颜色

### 4️⃣ 添加图例和标签

- **图例**：清晰标识每条折线代表的变量
- **标签**：添加轴标签和单位
- **标题**：为图表添加描述性标题

## 🔧 故障排除

### 常见问题

#### ❓ 图表不显示数据

**可能原因**：
- 标签未正确绑定
- 数据源未连接
- 时间范围设置不当

**解决方案**：
1. 检查标签绑定是否正确
2. 确认设备连接状态
3. 调整时间范围设置

---

#### ❓ 图表刷新缓慢

**可能原因**：
- 数据量过大
- 刷新率过高
- 系统资源不足

**解决方案**：
1. 减少显示的数据点数量
2. 降低刷新频率
3. 优化系统性能

---

#### ❓ 图表显示异常

**可能原因**：
- 数据类型不匹配
- 数值超出范围
- 配置错误

**解决方案**：
1. 检查数据类型是否正确
2. 确认数值在合理范围内
3. 检查图表配置
