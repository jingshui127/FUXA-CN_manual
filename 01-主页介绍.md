# 🎯 FUXA 官方文档 - 欢迎使用 FUXA

---

## 📖 什么是 FUXA？

FUXA 是一款强大的**基于 Web** 的软件，能够快速构建和部署可扩展的 SCADA、HMI、仪表板或 IIoT 系统。

使用 FUXA，您可以：
- ✅ 为您的机器创建现代化的过程可视化界面
- ✅ 实现个性化设计
- ✅ 实时数据显示
- ✅ 控制自动化工业设备

![](images/fuxa-editor.png)

---

## 🔌 通信协议支持

FUXA 支持常用的工业通信标准，包括西门子 S7 协议和 OPC UA，允许连接第三方 OPC 服务器。支持的通信标准列表可以通过开发额外的驱动程序来扩展。

### 支持的协议包括：

| 协议类型 | 说明 |
|---------|------|
| **OPC UA** | 工业自动化标准协议 |
| **Siemens S7** | 西门子 PLC 通信协议（支持 200/300/400/1200/1500 系列） |
| **Modbus RTU/TCP** | 通用工业通信协议 |
| **BACnet IP** | 楼宇自动化协议 |
| **MQTT** | 物联网消息协议 |
| **Ethernet/IP** | Allen Bradley 设备协议 |
| **WebAPI** | Web 接口集成 |

---

## 💻 软件架构

FUXA 基于 **Node.js** 运行时开发，您可以通过 Web 浏览器访问编辑器来创建应用程序（SCADA/HMI/仪表板），作为客户端运行可视化界面。

---

## 🖥️ 用户界面

FUXA 由两个不同的视图组成：

### 1️⃣ FUXA 编辑器（Editor）
用于编辑和设计项目

### 2️⃣ FUXA 视图（View）
用于展示可视化项目的最终产品

![](images/fuxa-home.png)

## ✨ 核心特性

### 🆓 无运行时许可证限制
FUXA 不需要任何运行时许可证！您可以构建任意数量和规模的 HMI 项目，无需担心运行时环境的额外许可费用。

### 🌐 跨平台全栈解决方案
该系统设计为在各种操作系统中运行，安装简便。

| 技术栈 | 说明 |
|--------|------|
| **后端** | Node.js 开发 |
| **前端** | 可扩展的 HTML5 Web 前端 |
| **技术** | HTML5、CSS、JavaScript、Angular、SVG |
| **兼容性** | 支持所有现代浏览器 |

### 🚀 灵活部署
- 支持 Windows、Mac、Linux 全平台
- 支持 Docker 容器化部署
- 支持本地安装和云端部署

---

## 🎯 适用场景

FUXA 适用于以下场景：

- 🏭 **工业自动化**：工厂设备监控、生产线管理
- 🌡️ **过程控制**：温度、压力、流量等参数监控
- 📊 **数据可视化**：实时数据仪表板、趋势分析
- 🏠 **楼宇自动化**：空调、照明、安防系统集成
- 🌐 **物联网应用**：IIoT 设备数据采集与展示
- 📱 **远程监控**：多终端访问、移动端支持

## 💡 快速开始

### 第一步：安装 FUXA

选择适合您的安装方式：

1. **Docker 部署**（推荐生产环境）
2. **NPM 安装**（适合开发者）
3. **预编译版本**（最简单）
4. **源码编译**（高级用户）

详细安装教程请参考：[安装和运行指南](03-安装和运行指南.md)

### 第二步：访问 FUXA

安装完成后，在浏览器中访问：
```
http://localhost:1881
```

默认登录账号：
- 用户名：`admin`
- 密码：`admin`

### 第三步：创建项目

1. 进入编辑器界面
2. 连接您的设备
3. 创建标签
4. 设计可视化界面
5. 部署运行
