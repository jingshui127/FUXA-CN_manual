# 💾 FUXA 项目管理教程

---

## 📋 概述

项目管理功能允许您创建、保存、加载和导出 FUXA 项目，方便项目备份和迁移。

---

## 🎯 创建项目

### 创建新项目

您可以创建一个新项目，当前项目将被新的空项目覆盖（新项目只包含一个空的 MainView）。

![创建项目](images/fuxa-project.png)

### 操作步骤

1. 打开 FUXA 编辑器
2. 点击 **"File"**（文件）→ **"New Project"**（新建项目）
3. 确认创建新项目

### 注意事项

⚠️ **警告**：创建新项目将覆盖当前项目，请确保已备份重要数据！

---

## 💾 保存项目

### 自动保存

项目会在每次设置更改（对话框确认）后自动保存到内部数据库，例如编辑设备、标签、图表等。

视图会在选择其他视图或离开编辑器时自动保存。

### 手动保存

#### 保存项目

通过 **"Save Project"**（保存项目）可以强制执行内部保存过程。

**操作步骤**：
1. 点击 **"File"**（文件）→ **"Save Project"**（保存项目）
2. 等待保存完成

#### 另存为

通过 **"Save Project As…"**（项目另存为）可以将整个项目导出为 JSON 格式的文件（MyProject.json），这对于项目备份非常有用。

**操作步骤**：
1. 点击 **"File"**（文件）→ **"Save Project As…"**（项目另存为）
2. 选择保存位置
3. 输入文件名
4. 点击保存

### 项目文件格式

```json
{
  "name": "MyProject",
  "version": "1.0.0",
  "created": "2024-01-01T00:00:00.000Z",
  "modified": "2024-01-01T00:00:00.000Z",
  "devices": [],
  "tags": [],
  "views": [],
  "charts": [],
  "alarms": [],
  "scripts": [],
  "settings": {}
}
```

---

## 📂 打开项目

### 从文件打开项目

您可以从导出的文件（MyProject.json）打开项目。

**操作步骤**：
1. 点击 **"File"**（文件）→ **"Open Project"**（打开项目）
2. 选择项目文件
3. 点击打开

### 注意事项

⚠️ **警告**：打开项目将覆盖当前项目，请确保已备份重要数据！

---

## 🔄 项目备份与恢复

### 备份策略

#### 定期备份

建议定期备份项目，特别是在进行重大更改之前。

**备份步骤**：
1. 点击 **"File"**（文件）→ **"Save Project As…"**（项目另存为）
2. 选择备份目录
3. 使用带日期的文件名（例如：MyProject_20240101.json）
4. 点击保存

#### 版本控制

使用版本控制系统（如 Git）管理项目文件。

**示例**：
```bash
# 初始化 Git 仓库
git init

# 添加项目文件
git add MyProject.json

# 提交更改
git commit -m "Initial commit"

# 创建标签
git tag v1.0.0
```

### 恢复项目

#### 从备份恢复

**恢复步骤**：
1. 点击 **"File"**（文件）→ **"Open Project"**（打开项目）
2. 选择备份文件
3. 点击打开

#### 从版本控制恢复

**示例**：
```bash
# 查看提交历史
git log

# 恢复到特定版本
git checkout v1.0.0
```

## 💡 完整示例

### 示例 1：创建新项目

```javascript
// 创建新项目
1. 打开 FUXA 编辑器
2. 点击 File → New Project
3. 确认创建新项目
4. 开始配置设备和标签
5. 自动保存或手动保存
```

### 示例 2：备份项目

```javascript
// 备份项目
1. 点击 File → Save Project As…
2. 选择备份目录：C:\Backups\FUXA
3. 输入文件名：MyProject_20240101.json
4. 点击保存
5. 确认备份成功
```

### 示例 3：恢复项目

```javascript
// 恢复项目
1. 点击 File → Open Project
2. 选择备份文件：C:\Backups\FUXA\MyProject_20240101.json
3. 点击打开
4. 确认恢复成功
5. 验证项目配置
```

## 🔧 故障排除

### 常见问题

#### ❓ 项目无法保存

**可能原因**：
- 磁盘空间不足
- 文件权限问题
- 数据库连接失败

**解决方案**：
1. 检查磁盘空间
2. 验证文件权限
3. 确认数据库连接

---

#### ❓ 项目无法打开

**可能原因**：
- 文件损坏
- 文件格式不正确
- 版本不兼容

**解决方案**：
1. 检查文件完整性
2. 验证文件格式
3. 确认版本兼容性

---

#### ❓ 自动保存不工作

**可能原因**：
- 自动保存未启用
- 数据库连接失败
- 权限不足

**解决方案**：
1. 检查自动保存设置
2. 验证数据库连接
3. 确认用户权限
