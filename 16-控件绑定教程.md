## 💡 控件类型说明

### 输出控件（Output Control）

- **用途**：显示标签的当前值
- **特点**：只读，不能修改
- **应用场景**：
  - 显示温度、压力等传感器值
  - 显示设备状态
  - 显示计算结果

### 输入控件（Input Control）

- **用途**：允许用户输入值并写入标签
- **特点**：可编辑，支持数值和字符串
- **应用场景**：
  - 设置目标温度
  - 输入设备参数
  - 修改控制值

### 选择控件（Select Control）

- **用途**：从预定义选项中选择值
- **特点**：下拉菜单，限制输入范围
- **应用场景**：
  - 选择设备模式
  - 选择操作模式
  - 选择预设参数

### 滑块控件（Slider Control）

- **用途**：通过滑块调整数值
- **特点**：直观的数值调整
- **应用场景**：
  - 调整速度
  - 调整亮度
  - 调整开度

## 📊 控件属性配置

### 通用属性

| 属性 | 说明 | 示例 |
|------|------|------|
| **标签** | 控件显示的标签 | 温度 |
| **标签 ID** | 绑定的标签 ID | temperature |

| 属性 | 说明 | 推荐值 |

## 💡 使用示例

### 示例 1：温度显示与控制

```javascript
// 输出控件：显示当前温度
{
  label: '当前温度',
  tagId: 'currentTemperature',
  unit: '°C',
  precision: 1
}

// 输入控件：设置目标温度
{
  label: '目标温度',
  tagId: 'targetTemperature',
  unit: '°C',
  min: 0,
  max: 100,
  step: 1
}
```

### 示例 2：设备模式选择

```javascript
// 选择控件：选择设备模式
{
  label: '设备模式',
  tagId: 'deviceMode',
  options: [
    { label: '自动', value: 0 },
    { label: '手动', value: 1 },
    { label: '停止', value: 2 }
  ],
  default: 0
}
```

### 示例 3：速度控制

```javascript
// 滑块控件：调整速度
{
  label: '速度',
  tagId: 'motorSpeed',
  min: 0,
  max: 100,
  step: 1,
  showValue: true,
  unit: '%'
}
```

## 🔧 故障排除

### 常见问题

#### ❓ 控件不显示数据

**可能原因**：
- 标签 ID 错误
- 设备未连接
- 数据类型不匹配

**解决方案**：
1. 检查标签 ID 是否正确
2. 确认设备连接状态
3. 验证数据类型是否匹配

---

#### ❓ 输入值无法写入

**可能原因**：
- 权限不足
- 标签为只读
- 值超出范围

**解决方案**：
1. 检查用户权限
2. 确认标签是否可写
3. 验证值是否在有效范围内

---

#### ❓ 控件样式异常

**可能原因**：
- CSS 样式冲突
- 浏览器兼容性问题
- 配置错误

**解决方案**：
1. 检查 CSS 样式
2. 尝试使用不同的浏览器
3. 检查控件配置

## 📚 最佳实践

### 1️⃣ 数据验证

```javascript
// 输入验证
function validateInput(value, min, max) {
  if (value < min || value > max) {
    alert('值超出范围！');
    return false;
  }
  return true;
}
```

### 2️⃣ 单位一致性

- 确保所有相关控件使用相同的单位
- 在标签中明确显示单位
- 提供单位转换功能（如果需要）

### 3️⃣ 用户友好性

- 提供清晰的标签和说明
- 使用合理的默认值
- 提供输入验证和错误提示

### 4️⃣ 性能优化

- 避免过于频繁的数据更新
- 使用节流函数限制更新频率
- 优化控件渲染性能
