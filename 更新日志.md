# 📋 FUXA 版本更新日志

> 查看 FUXA 软件从 v1.0.0 到 v1.2.9 的所有版本更新内容和功能改进。

---

## 📊 版本概览

### 📈 版本统计

从 v1.0.0 到 v1.2.9，FUXA 共发布了 **37 个主要版本**，包含了数百项功能更新和问题修复。

### 🎯 版本演进趋势

- **v1.2.x 系列**：主要关注 API 增强、图表优化、设备协议扩展和国际化支持
- **v1.1.x 系列**：主要关注基础功能完善、用户体验改进和核心稳定性提升
- **v1.0.x 系列**：主要关注初始功能开发、协议支持和基础架构建设

### 🏆 主要里程碑

**v1.2.0** 是重要的架构升级版本，引入了多视图容器、Angular 15 升级和 Electron 构建支持。

**v1.1.01** 是重大重构版本，标签 ID 使用 GUID 重构（不兼容旧版本），引入多视图容器和主题切换。
