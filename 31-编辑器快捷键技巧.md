# ⌨️ FUXA 编辑器快捷键技巧

---

## 📋 概述

掌握 FUXA 编辑器的键盘快捷键可以大大提高您的工作效率。以下是常用的快捷键列表。

---

## 🎯 编辑器键盘快捷键

### 旋转操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl + 左 / 右** | 旋转选中项 | 旋转选中的元素 |
| **Ctrl + Shift + 左 / 右** | 大步旋转 | 以较大步长旋转选中的元素 |

### 选择操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Shift + O / P** | 选择前一项 / 后一项 | 选择上一个 / 下一个元素 |
| **Tab / Shift + Tab** | 选择前一项 / 后一项 | 选择上一个 / 下一个元素 |
| **Ctrl + A** | 选择所有项 | 选择视图中的所有元素 |

### 缩放操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl + 上 / 下** | 中心缩放 | 以视图中心为基准放大 / 缩小 |
| **Shift + 滚轮** | 鼠标位置缩放 | 以鼠标位置为基准放大 / 缩小 |

### 撤销 / 重做

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl + Z** | 撤销 | 撤销上一步操作 |
| **Ctrl + Y** | 重做 | 重做上一步撤销的操作 |

### 移动操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Shift + 上 / 下 / 左 / 右** | 移动选中项 | 移动选中的元素 |
| **Shift + 鼠标调整大小** | 锁定宽高 | 在调整大小时锁定宽高比例 |

### 复制 / 粘贴

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl + X** | 剪切选中项 | 剪切选中的元素 |
| **Ctrl + C** | 复制选中项 | 复制选中的元素 |
| **Ctrl + V** | 粘贴选中项 | 粘贴剪贴板中的元素 |
| **Ctrl + D** | 复制选中项 | 复制选中的元素 |

### 分组操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Ctrl + G** | 分组 / 取消分组 | 将选中的元素分组或取消分组 |

### 绘图操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Shift + 绘制线条** | 线条渐变 | 绘制水平、垂直或 45° 对角线 |

## 💡 使用技巧

### 1️⃣ 快速导航

使用 **Tab** 或 **Shift + Tab** 快速在元素之间切换选择。

### 2️⃣ 精确旋转

使用 **Ctrl + Shift + 左 / 右** 进行大步旋转，快速调整角度。

### 3️⃣ 保持比例

使用 **Shift + 鼠标调整大小** 在调整大小时锁定宽高比例。

### 4️⃣ 快速复制

使用 **Ctrl + D** 快速复制选中的元素，提高工作效率。

### 5️⃣ 精确移动

使用 **Shift + 方向键** 精确移动选中的元素。

### 6️⃣ 快速缩放

使用 **Shift + 滚轮** 以鼠标位置为基准进行缩放，更加精确。

## 🎯 常用场景

### 场景 1：创建仪表盘

1. 使用 **Ctrl + A** 选择所有元素
2. 使用 **Ctrl + G** 将元素分组
3. 使用 **Ctrl + D** 复制分组
4. 使用 **Shift + 方向键** 调整位置

### 场景 2：调整布局

1. 使用 **Tab** 选择元素
2. 使用 **Shift + 鼠标调整大小** 调整大小
3. 使用 **Ctrl + 左 / 右** 旋转元素
4. 使用 **Shift + 滚轮** 缩放视图

### 场景 3：快速编辑

1. 使用 **Ctrl + C / V** 复制 / 粘贴元素
2. 使用 **Ctrl + Z / Y** 撤销 / 重做操作
3. 使用 **Ctrl + X** 剪切元素
4. 使用 **Ctrl + D** 快速复制

## 🔧 故障排除

### 常见问题

#### ❓ 快捷键不工作

**可能原因**：
- 输入法冲突
- 浏览器快捷键冲突
- 快捷键被禁用

**解决方案**：
1. 切换到英文输入法
2. 检查浏览器快捷键设置
3. 确认快捷键是否启用

---

#### ❓ 无法旋转元素

**可能原因**：
- 元素不支持旋转
- 快捷键冲突
- 元素被锁定

**解决方案**：
1. 确认元素是否支持旋转
2. 检查快捷键是否冲突
3. 解锁元素

---

#### ❓ 无法复制元素

**可能原因**：
- 元素被锁定
- 权限不足
- 快捷键冲突

**解决方案**：
1. 解锁元素
2. 确认用户权限
3. 检查快捷键是否冲突
